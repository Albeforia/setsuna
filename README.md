# setsuna
A tiny graphics engine.

# Building
## Prerequisites
- cmake (>=3.0)
- glad (OpenGL core profile >= 4.5 + ARB_sparse_texture)
- glm
- c++17 compiler

## Extra
If you want to build **setsuna_loaders**,

- stbi

are required.

If you want to build **applications**,

- GLFW

are required.
